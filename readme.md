# Laboration 3 - CSS Preprocessorer

En laboration inom CSS-preprocessor där jag skapar kod genom SCSS/Sass istället för CSS. Jag skapade partials med filändelse-namnet scss. Inom de olika partials lade jag in regler för komponenter (knappar), variabler för att lagra färger och fonter, samt layout och mixins i separata. Stuktur på sidorna genom HTML, samt hamburgermeny till mindre skärmar. Två bilder lades till på startsidan.

Inom uppgiften använde jag:
- Variabler, color functions, mixins, nesting av kod, @extend, @if/else.
